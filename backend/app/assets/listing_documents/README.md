Canonical blank PDF templates for the donor compliance workflow live in this directory.

- `decontamination-template-2026-03-26.pdf`
- `liability-release-template-2026-03-26.pdf`

The donor-facing modal editor writes structured form data, and the backend generates the saved per-listing PDF artifact from that data. These versioned blank PDFs are kept in-repo as the source template references for the workflow.
